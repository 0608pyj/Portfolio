# Portfolio

저는 언어의 힘으로 소통을 이끌고, 연대의 가치로 학생과 함께 성장하는 국어교사가 되고자 합니다. 

국어국문학을 전공하며 언어가 단순한 의사소통의 도구가 아니라, 사람들 간의 관계를 형성하고, 사고와 감정을 나누는 중요한 매개체임을 깨달았습니다. 특히, 화용론을 공부하면서 말의 의미가 상황과 맥락에 따라 변화하는 과정이 매우 흥미로웠습니다. 같은 말을 하더라도 누가, 언제, 어디서, 어떤 의도로 말하느냐에 따라 의미가 달라지는 점은 언어의 깊이를 더해주었습니다.

1학년 때, 문과대학 학생회 연대소통부에서 활동을 시작한 저는 의견이 갈리는 상황에서 중재 역할을 맡았습니다. 문과대학 체전, 농민학생연대활동, e스포츠 대회를 기획하며 다양한 의견을 조율하고 갈등을 해결해 나갔습니다. 특히, 서로 다른 의견을 존중하고, 상대의 입장에서 생각하는 법을 배운 경험은 국어교사로서 제게 큰 자산이 될 것입니다. 이러한 경험을 통해 소통의 진정성은 상대방의 입장을 깊이 이해하고 공감하는 것이라는 사실을 깨달았습니다.

저는 교실에서 학생들의 개별적인 목소리와 생각을 존중하며, 그들이 자유롭게 의견을 나누고 자기 자신을 표현할 수 있는 환경을 만들어가고 싶습니다. 국어를 통해 학생들이 소통하고, 사고를 확장하며, 서로 다른 관점을 이해하는 능력을 기를 수 있도록 돕겠습니다. 진심 어린 소통을 통해 학생들이 성장하는 모습을 지켜보는 것이 제 꿈입니다.
