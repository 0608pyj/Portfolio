# Portfolio
제가 하고 싶은 것은 언어의 힘으로 소통을 이끌고, 연대의 힘으로 학생과 함께 성장하는 국어교사입니다. 
국어국문학을 전공하며 텍스트 속 의미를 깊이 있게 이해하는 동시에, 사람과사람 사이의 소통이 얼마나 중요한지 깨달아 왔습니다. 1학년 때 문과대학 학생회 연대소통부 부원으로 활동을 시작해, 현재는 연대소통부 부장으로서 구성원 간의 연결과 협력을 이끌고 있습니다. 문과대학 체전, 농민학생연대활동, e스포츠 대회를 기획하고 총괄하며 다양한 사람의 의견을 조율하고 하나의 방향으로 모아가는 경험을 쌓았습니다. 특히 여러 이해관계 속에서 소통을 통해 문제를 해결하고, 모두가 만족할 수 있는 결과를 만들어내는 과정에서 큰 보람을 느꼈습니다. 
이를 통해 제가 가장 중요하게 생각하게 된 가치는 '소통'입니다. 단순히 의견을 전달하는 것을 넘어, 상대의 입장을 이해하고 공감하는 태도가 조직을 움직인다고 믿습니다. 이러한 경험은 앞으로 교사가 되어 학생들과 관계를 맺고, 각자의 생각을 존중하며 성장할 수 있는 교실을 만드는 데 중요한 기반이 될 것입니다. 저는 언어를 통해 학생들과 진심으로 소통하며, 함께 배우고 성장하는 국어교사가 되고자 합니다.
